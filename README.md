# Macro LLM Tournament

This repository contains a runnable 200-household LLM economy. Real SCE histories
supply household priors, matched SCF records supply continuous financial states,
GPT-5.5 writes conditional household policies in dollars, and ordinary code settles
budgets and counterparties. A small producer turns household demand into output,
inventory, employment, wages, and family income before the households decide again.

The latest anchor-free run is a clean negative on forecast value. Across four
historical monthly origins, forecast consumption growth has RMSE `0.782` percentage
points and gets one direction right. The origin-visible PCE drift supplied to every
household does substantially better: RMSE `0.243`, correlation `0.974`, and 4/4
direction. The economy is genuinely generated by household actions, but the household
layer has not yet added signal beyond the information in its prompt.

## Active Economy

```text
200 anonymized SCE histories + matched SCF financial states
                              +
                 public information known at the origin
                              |
                              v
                one isolated GPT-5.5 call per household
                              |
                              v
       beliefs + employed/not-employed policies in nominal dollars
                              |
                              v
          household budgets, credit, and settlement
                              |
                              v
                    population-weighted demand
                              |
                              v
     producer output + inventory -> employment + wages -> family income
                              |
                              v
               fresh period-two household decisions
```

Each rolling origin receives the latest SCE observations that would have been public
at that date. The public information card includes recent aggregate spending as
context, but it does not add that growth to household spending before the LLM acts.
The producer is deliberately mechanical. There are no LLM firms or banks.

## Run It

```bash
make ecology-history
make ecology-current-replay
make ecology-retrospective-replay
make ecology-feedback-replay
make ecology-observability
make check
make test
```

Fresh model calls use Codex CLI only:

```bash
make ecology-live-200
make ecology-retrospective-live
make ecology-feedback-live
```

Every model call runs in an empty directory without local-file, shell, web, browser,
app, memory, plugin, or subagent access. Replays bind the public history, prompt,
cards, accepted payloads, parent artifacts, source inputs, executable source, and
economic result.

## Evidence Boundary

- January-April 2026 is retrospective development evidence and may be in model
  knowledge.
- The July 2026 origin is frozen for August and remains unscored.
- Household budgets, goods inventory, bank stocks, and named counterparty flows
  reconcile. A firm balance sheet and full external sector are not modeled.
- V22's apparent 4/4 directional result is retired. It inherited origin-visible PCE
  growth in the household spending baseline before the model acted.
- V23 removes that anchor. Its household forecast currently loses to the visible PCE
  context on level, timing, and direction.
